Thesis Progress Submission

For Students
1

Prepare the progress report and save it as a PDF file using the filename format StudentID-Name-Year-Semester (e.g., 6830000-Sofia La-2026-1).
Download the progress report template here.

2

Combine the progress report and Seminar Passport into a single PDF file.

3

Submit the file directly to Major Advisor.

For Advisors & Program Directors
1

Review and evaluate the submitted progress report.

2

Sign the document in both the progress report and Seminar Passport sections.

3

Upload the signed PDF file to the corresponding folder in the shared drive:
Thesis Progress Folder

4

Log in to the Online Thesis Advisor system and submit the evaluation through the following workflow:

Online Thesis Advisor (อนุมัติ บฑ. ทุกประเภท)
 ↳ Grading for Thesis/Thematic Paper
   ↳ Evaluation of thesis for each term

Note: For students who have not yet completed the proposal examination, the Program Director is responsible for submitting their thesis/dissertation grade.
